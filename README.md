# Full-Stack Orders Application

This is a full-stack web application for managing orders. It includes:
- A **Backend** powered by **Node.js** and **MongoDB**.
- A **Frontend** built with **React** and styled using **Tailwind CSS**.

## Features
- User authentication with cookies.
- Fetch and display orders with menu items, quantities, total amounts, and order statuses.
- Responsive and user-friendly UI.
- Backend API with endpoints for creating and fetching orders.

---

## Getting Started

Follow these instructions to set up and run the project locally.

### Prerequisites
Make sure you have the following installed on your machine:
- **Node.js** (version >= 16.x)
- **npm** or **yarn**
- **MongoDB Atlas** or a local MongoDB instance
- **Git**
